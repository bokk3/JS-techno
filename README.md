# JS-techno

Live-coded acid techno performance system built in Strudel.

This project is a **live audiovisual memorial set** dedicated to the legendary 90’s Belgian nightlife scene — clubs that shaped a generation of dance culture and later disappeared.

Each performance is a reconstruction of memory:
not replayed music, but **rebuilt moments through code in real time**.

No DAWs. No fixed tracks. Only patterns, synthesis, and live manipulation.

---

## 🕯️ Concept

“Een eerbetoon aan de verdwenen danscultuur.”

This project treats each track as a **living memory of a club space**:

* not songs, but *rooms*
* not arrangements, but *states*
* not playback, but *reconstruction*

The set evolves like walking through a fading club at different hours of the night — from peak-time euphoria to empty after-hours decay.

## 🎛️ Live Coding System

All music is generated in real time using **Strudel**, a browser-based live coding environment for pattern-based music.

👉 Try it yourself here:
https://strudel.cc

You can:

* write patterns live
* modify rhythm instantly
* control synthesis parameters
* perform music like code

---

## 👁️ Live Visualisation

Visuals are not separate from the music — they are generated as part of the same system.

Each sound layer can drive real-time visual feedback such as:

* **pianoroll structures** (melodic shape visualization)
* **oscilloscope views** (waveform movement)
* **spectrum analysis** (frequency energy)
* pattern-driven motion and rhythm visualization

In performance, this creates a direct link between:

> sound → pattern → image → memory

The visuals act as a “ghost layer” of the club experience — a digital residue of sound made visible.

Future versions will expand into full generative audiovisual systems (Three.js / shader-based environments / reactive lighting states).
